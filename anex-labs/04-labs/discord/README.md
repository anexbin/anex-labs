# Discord Security Research Track

This folder contains structured labs, experiments, and documentation for one-year deep Discord pentesting practice, focused on building skills for Bugcrowd and SaaS security.

## Index of Labs

1. **01-roles-permissions** - Explore Discord roles, permissions, and potential misconfigurations.
2. **02-channel-overrides** - Experiment with channel-specific permission overrides and edge cases.
3. **03-bots-intents** - Test bot permissions, intents, and security implications.
4. **04-oauth2-testing** - Learn about OAuth2 flows, token handling, and potential flaws.
5. **05-api-rate-limits** - Understand Discord API rate limiting, abuse scenarios, and bypass attempts.
6. **06-automation-scripts** - Develop scripts for safe automation, testing, and vulnerability checks.
7. **07-failure-patterns** - Document and analyze patterns of failure or misconfiguration for research.
